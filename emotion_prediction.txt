ðŸ’¡ Functionality Summary

1. Text Input

* User enters a text.
* Model: `j-hartmann/emotion-english-distilroberta-base`.
* Predicts one of the following emotions:

* Anger, Disgust, Fear, Joy, Neutral, Sadness, Surprise.
* Output: Emotion label and prediction confidence.

2. Voice Input

* Uses `speech_recognition` to capture and transcribe voice.
* Transcribed text is passed through the same text emotion model.
* Output: Recognized text, predicted emotion, and confidence.

3. Emoji Input

* User inputs a single emoji character.
* Model: `cardiffnlp/twitter-roberta-base-emoji`.
* Maps top emoji prediction to an emotion using a custom dictionary.
* Output: Closest matching emoji, corresponding emotion, and confidence.
